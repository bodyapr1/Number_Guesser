# Guess the Number 🎯

Гра "Вгадай число" - це захоплююча логічна гра, де гравець повинен відгадати випадково згенероване число з унікальними або повторюваними цифрами, отримуючи підказки після кожної спроби.

## 🎮 Про гру

Гравець вводить комбінацію цифр, а гра показує підказки кольорами:
- **Зелений** 🟢 - цифра на правильній позиції
- **Жовтий** 🟡 - цифра є в числі, але на іншій позиції
- **Сірий** ⚪ - цифри немає в загаданому числі

## ✨ Особливості

- ⏱️ **Таймер** - відстеження часу гри
- 🎨 **Кольорові підказки** - інтуїтивно зрозумілий зворотній зв'язок
- ⚙️ **Налаштування гри**:
  - Довжина числа (1-10 цифр)
  - Кількість спроб (1-10)
  - Можливість повторення цифр
- 🌓 **Темна/світла тема** - комфортна гра в будь-який час
- 📱 **Адаптивний дизайн** - гра працює на всіх пристроях

## 🚀 Як грати

1. Введіть комбінацію цифр у поле введення
2. Натисніть "Try" або клавішу Enter
3. Отримайте кольорову підказку:
   - 🟢 Зелений - цифра на своєму місці
   - 🟡 Жовтий - цифра є в числі, але не на цьому місці
   - ⚪ Сірий - цифри немає в числі
4. Використовуйте підказки для наступних спроб
5. Вгадайте число, поки не закінчилися спроби!
